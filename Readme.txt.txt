Flask MySQL CRUD Application

This is a simple CRUD (Create, Read, Update, Delete) web application
developed using Flask (Python) and MySQL.

Technologies Used:
- Python
- Flask
- MySQL
- HTML

Features:
- Add student records
- View student records
- Edit student records
- Delete student records

How to Run:
1. Install required packages:
   pip install -r requirements.txt

2. Update MySQL credentials in app.py.

3. Run the application:
   python app.py

4. Open a browser and visit:
   http://127.0.0.1:5000

Note:
This application runs locally using the Flask development server
